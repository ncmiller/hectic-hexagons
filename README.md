# Hectic Hexagons

A clone of Hexic HD, written in C using the SDL2 library.

### Install Dependencies

MacOS:

```sh
brew install sdl2
```

Ubuntu:

```sh
sudo apt-get install libsdl2-dev
```

### Build

```sh
./build.sh
```

### Run

After building, to run the game executable:

```
./build/hectic-hexagons
```
